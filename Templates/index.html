An API for managing tourist attractions in Slovakia.

Allows developers to interact with a database to perform various operations such as searching, adding, updating, and removing tourist attractions.

Here's a simple description of these functionalities:

Add Attraction:

Users can use this functionality to add new tourist attractions to the database. They would submit the necessary information, including attraction name, location url address (Google Maps, etc...), region, nearest city and type (e.g., cultural, recreational), .

Search Attractions:

This feature enables users to search for tourist attractions in Slovakia based on various criteria such as location, type for example: historical sites, natural landmarks, region, city or name. The API would accept search parameters and return a list of matching attractions with relevant details.

Update Attraction:

This feature allows users to modify the details of existing tourist attractions. Users would provide the attraction's name along with the updated information, such as changed name, or location. The API would update the attraction accordingly.

Remove Attraction:

This functionality permits users to remove tourist attractions from the database. Users would specify the attraction's name, and the API would remove the corresponding attraction.
