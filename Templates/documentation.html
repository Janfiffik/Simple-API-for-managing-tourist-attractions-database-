Documentation

Get all Attractions:

For getting all attractions use: http://127.0.0.1:5000/all

Get all types, regions or cities:

Possible column names: REGION, CITY, TYPE

Example:

http://127.0.0.1:5000/column_data/CITY

Filter data of Attractions by:

you can choose Columns:

Column names: NAME, REGION, CITY, TYPE.

Example:

http://127.0.0.1:5000/column/REGION/name/KOSICE

Add Attraction:

For adding new attraction use:

http://127.0.0.1:5000/add

In to body of request you need to add:

{"NAME": "name of the tourist attraction", "LOC": "location(google map link or something similar)", "REGION": "name of the region", "CITY": "nearest city or town"}

Update Attraction:

http://127.0.0.1:5000/update/name of attraction

In to the body of request you can specify which value you want update example: {"REGION": "Other region than original"}

Remove Attraction:

http://127.0.0.1:5000//remove_attraction/attraction name
